# Desktop application for internal work for restaurant

Desktop application that provides tools and functions for owners (administration work), waiters (create table, create order, update table with new order, pay table) and sub-application for inside the kitchen (view orders with priorities in clean and understandable way).

## Technologies

- **TypeScript** – superset of JS, adding static typing
- **Electron** – JS library that allows creation of desktop applications with the use of frontend technologies like javascript, typescript, react, vue, angular etc.
- **Vite** – tool for building and developing frontend applications
- **React** – JS library for creating user interfaces
- **React ecosystem** - Set of libraries from React, for example: react-intl, react-router, axios, react-icons, react-toast etc.
- **JSDoc** – Files and code documentation for easier navigation and maintanance

## How to run

In order to run this application use the steps below:

1. Open terminal in the environment you are using (can be VS code or other) and run **npm install**.
2. After first command, run **npm run dev** to start the application.

## Notice

You must have the backend on running in order to use this application and its features.
